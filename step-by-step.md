## Шаг 1. Выбор роутера

Для выборочного туннелирования нужен роутер, поддерживающий WireGuard или OpenVPN на уровне прошивки.

### Keenetic (проще всего для новичка)

Подходят любые модели с KeeneticOS 4.x/5.x и компонентом WireGuard или OpenVPN:

| Модель | Для чего подходит |
|---|---|
| Keenetic Carrier KN-1721 | Базовый, недорогой вариант |
| Keenetic Giga KN-1010/1011/1012 | Мощнее, для быстрых тарифов |
| Keenetic Hopper, Viva, Extra, Titan | Любая свежая модель с прошивкой 4.x+ |

Проверить наличие нужного компонента: **Управление - Параметры системы - Изменить набор компонентов**.

### Xiaomi / другие под OpenWrt

Подходят модели, которые официально поддерживаются в [таблице устройств OpenWrt](https://openwrt.org/toh/start). Обязательно проверяйте **ревизию** (v1, v2 и т.д.) - одна и та же модель разных ревизий может прошиваться по-разному или не прошиваться вообще.

Проверенные варианты:
- Xiaomi Router AX3000T
- Xiaomi Mi Router 4A Gigabit
- Mi Router 4C

> Купить Keenetic можно в DNS, М.Видео или Яндекс.Маркете. Xiaomi - там же или на Wildberries.

## Шаг 2. Аренда VPS

### Что такое VPS простыми словами

VPS (Virtual Private Server) - это маленький кусочек настоящего сервера в зарубежном дата-центре, который вы арендуете за фиксированную плату. Он постоянно работает, имеет свой IP-адрес и выдаёт вам три строчки для входа: IP, логин и пароль.

### Где арендовать

Популярные площадки с серверами в Европе:

| Площадка | Особенности |
|---|---|
| [VDSina](https://vdsina.com) | Хороший старт, есть локации в Европе |
| [ISHosting](https://ishosting.com) | Нидерланды, Германия |
| [Aeza](https://aeza.net) | Быстрые серверы, Европа |
| [4VPS](https://4vps.su) | Широкий выбор локаций |
| [Fornex](https://fornex.com) | Германия, Финляндия |
| [Timeweb Cloud](https://timeweb.cloud) | Российский провайдер с зарубежными локациями |
| [Beget](https://beget.com) | Надёжный провайдер |
| [FirstVDS](https://firstvds.ru) | Доступные тарифы |
| [Handyhost](https://handyhost.ru) | Удобная панель управления |

### Что выбирать при регистрации

1. **Локация** - Германия, Нидерланды, Финляндия. Хороший пинг из России.
2. **ОС** - Ubuntu 22.04 LTS или Debian 12. Самые стабильные и поддерживаемые.
3. **Тариф** - для домашнего использования хватит минимального. Старт от ~200 руб./мес ($2.10).
4. **Оплата** - большинство площадок принимают карты РФ, СБП, крипту.

### Что вы получаете после оплаты

На почту придёт письмо с тремя строками:
```
IP: 185.xxx.xxx.xxx
Login: root
Password: xXxxxxxxXx
```
Это всё, что нужно для следующего шага.

## Шаг 3. Установка туннеля на сервер

Есть два пути: быстрый через приложение Amnezia или ручной через OpenVPN-скрипт.

### 3.1 Вариант A: Amnezia (рекомендуется для новичков)

Amnezia - это приложение с графическим интерфейсом, которое само подключается к вашему VPS и само устанавливает на него нужный протокол (AmneziaWG, OpenVPN, WireGuard). Никакой командной строки.

**На компьютере или телефоне:**

1. Скачайте приложение с [amnezia.org](https://amnezia.org) (есть для Windows, macOS, Android, iOS).
2. Запустите и выберите **«Настроить свой сервер»**.
3. Введите данные из письма:
   - IP-адрес сервера
   - Логин (root)
   - Пароль
4. Нажмите **«Далее»**.
5. Выберите протокол - рекомендуем **AmneziaWG** (это WireGuard с обфускацией, устойчив к блокировкам).
6. Нажмите **«Установить»** и подождите 2-3 минуты.

Приложение само:
- подключится к серверу по SSH
- установит нужные пакеты
- сгенерирует ключи шифрования
- создаст конфиг.

### 3.2 Вариант B: OpenVPN вручную

Если хотите OpenVPN без Amnezia, используйте готовый скрипт-установщик.

**Подключение к серверу по SSH:**
- Windows: используйте [PuTTY](https://putty.org) или встроенный терминал (Win+R -> cmd -> `ssh root@ВАШ_IP`)
- macOS/Linux: откройте терминал и введите `ssh root@ВАШ_IP`

**Установка OpenVPN через скрипт:**

```bash
curl -O https://raw.githubusercontent.com/angristan/openvpn-install/master/openvpn-install.sh
chmod +x openvpn-install.sh
bash openvpn-install.sh
```

Скрипт задаст несколько вопросов:
- IP-адрес сервера - он определится автоматически, просто нажмите Enter
- Порт - оставьте по умолчанию (1194) или выберите 443
- Протокол - UDP рекомендуется для скорости
- DNS - выберите Cloudflare (1.1.1.1) или Google (8.8.8.8)
- Имя первого клиента - например, `home`

По завершении скрипт создаст файл `home.ovpn` в домашней директории сервера.

**Скачать файл на компьютер:**
```bash
# С Windows (через PowerShell):
scp root@ВАШ_IP:~/home.ovpn C:\Users\ВашИмя\Downloads\

# С macOS/Linux:
scp root@ВАШ_IP:~/home.ovpn ~/Downloads/
```

## Шаг 4. Получаем конфиг (.conf или .ovpn)

### Если использовали Amnezia

1. В приложении нажмите на ваш сервер.
2. Выберите **«Поделиться» -> «Экспорт конфигурации»**.
3. Выберите формат:
   - **AmneziaWG (.conf)** - для роутеров с поддержкой AmneziaWG
   - **WireGuard (.conf)** - для роутеров с обычным WireGuard
   - **OpenVPN (.ovpn)** - если устанавливали OpenVPN
4. Сохраните файл.

### Если использовали скрипт OpenVPN

Файл `home.ovpn` уже лежит у вас на компьютере после выполнения `scp` из предыдущего шага.

> Этот файл - ваш ключ. Не публикуйте его. Не отправляйте в облако без пароля.

## Шаг 5. Подключаем роутер Keenetic

### 5.1 Keenetic + AmneziaWG

#### Подготовка роутера

1. Обновите KeeneticOS до версии 4.2+ (**Управление - Обновление**).
2. **Управление - Параметры системы - Изменить набор компонентов** - установите компонент **WireGuard VPN**.
3. Роутер перезагрузится.

#### Импорт конфига

1. **Интернет - Другие подключения - WireGuard**.
2. Нажмите **«Загрузить из файла»** и выберите ваш `.conf` файл.
3. Подключение появится в списке с именем, которое было в файле (например, `de`).
4. Войдите в подключение, включите его, но **не ставьте** галочку «Использовать для выхода в интернет» - нам нужен не основной канал, а обходной.

#### Добавление ASC-параметров (обфускация AmneziaWG)

Keenetic принимает параметры обфускации через веб-консоль. Откройте браузер и замените в адресной строке `/dashboard` на `/a`.

В консоли введите:
```
show interface Wireguard0
```
Убедитесь, что это ваш конфиг.

Затем введите команду с параметрами из вашего `.conf` файла (строки Jc, Jmin, Jmax, S1, S2, H1-H4):
```
interface Wireguard0 wireguard asc Jc Jmin Jmax S1 S2 H1 H2 H3 H4
system configuration save
```

### 5.2 Keenetic + OpenVPN

1. **Управление - Параметры системы - Изменить набор компонентов** - установите **OpenVPN client**.
2. После перезагрузки: **Интернет - Другие подключения - VPN-подключения - Создать подключение**.
3. Тип - **OpenVPN**, загрузите `.ovpn` файл.
4. Включите подключение. Галочку **«Использовать для выхода в интернет» не ставить**.
5. Проверьте, что подключение поднялось: должны идти счётчики «Принято/Отправлено».

## Шаг 6. Подключаем роутер Xiaomi / OpenWrt

### 6.1 Прошивка OpenWrt

1. Зайдите на [openwrt.org/toh/start](https://openwrt.org/toh/start) и найдите вашу модель и **точную ревизию** (указана на наклейке под роутером).
2. Скачайте **factory-образ** для вашей версии OpenWrt (23.05.x или 24.10.x).
3. Войдите в веб-интерфейс Xiaomi (обычно `192.168.31.1` или `miwifi.com`).
4. Найдите раздел обновления прошивки и загрузите скачанный файл.
5. После перезагрузки зайдите на `192.168.1.1` - это LuCI, веб-интерфейс OpenWrt.
   - Логин: `root`
   - Пароль: пустой (сразу смените в **System - Administration**)

> Важно: некоторые модели Xiaomi требуют предварительной разблокировки SSH через приложение Mi WiFi. Проверьте инструкцию именно для вашей модели на странице OpenWrt.

### 6.2 OpenWrt + AmneziaWG

#### Установка пакетов

Подключитесь к роутеру по SSH (`ssh root@192.168.1.1`) и выполните:

```bash
opkg update
opkg install kmod-amneziawg amneziawg-tools luci-proto-amneziawg
reboot
```

> Если пакеты не найдены, добавьте репозиторий вручную по инструкции на [github.com/amnezia-vpn/amneziawg-openwrt](https://github.com/amnezia-vpn/amneziawg-openwrt).

#### Создание интерфейса в LuCI

1. **Network - Interfaces - Add new interface**.
2. Имя: `awg_de` (или любое удобное).
3. Протокол: **AmneziaWG**.
4. Заполните поля из вашего `.conf` файла:
   - **Private Key** - строка `PrivateKey` из секции `[Interface]`
   - **IP Addresses** - строка `Address` из секции `[Interface]`
5. В блоке **AmneziaWG Settings** введите параметры обфускации: Jc, Jmin, Jmax, S1, S2, H1, H2, H3, H4.
6. Вкладка **Peers - Add peer**:
   - **Public Key** - строка `PublicKey` из секции `[Peer]`
   - **Endpoint Host** - IP из строки `Endpoint`
   - **Endpoint Port** - порт из строки `Endpoint`
   - **Allowed IPs** - `0.0.0.0/0`
   - **Persistent Keepalive** - `25`
7. Вкладка **Firewall** - назначьте зону `wan`.
8. **Save & Apply**.

#### Проверка туннеля

```bash
awg show
ping -I awg_de 1.1.1.1 -c 3
```

Должен появиться интерфейс и пройти пинг.

### 6.3 OpenWrt + OpenVPN

#### Установка пакетов

```bash
opkg update
opkg install openvpn-openssl luci-app-openvpn
```

После установки в LuCI появится меню **VPN - OpenVPN**.

#### Настройка подключения

1. **VPN - OpenVPN - Add**, тип - **Custom configuration**.
2. Нажмите **Edit** и вставьте содержимое вашего `.ovpn` файла целиком.
3. Включите инстанс галочкой и нажмите **Save & Apply**.
4. **Network - Interfaces - Add new interface**:
   - Имя: `ovpn0`
   - Протокол: **Unmanaged**
   - Устройство: `tun0`
5. Вкладка **Firewall** - зона `wan`. Сохраните.

#### Проверка туннеля

```bash
ip addr show tun0
ping -I tun0 1.1.1.1 -c 3
```

## Шаг 7. Выборочный роутинг

Цель: обычный трафик идёт через провайдера, YouTube/Telegram/ChatGPT - через туннель.

### 7.1 Keenetic: загрузка маршрутов через .bat

Готовые IP-списки с командами статической маршрутизации для Keenetic лежат в репозитории [RockBlack-VPN/ip-address](https://github.com/RockBlack-VPN/ip-address).

Скачайте нужные `.bat` файлы из раздела **Releases**:
- `Global/Youtube/...youtube...bat`
- `Global/Telegram/telegram.bat`
- `Global/OpenAI/chatgpt.bat`

Загрузка в роутер:
1. **Сетевые правила - Маршрутизация - Загрузить из файла**.
2. В поле **Интерфейс** выберите ваше VPN-подключение (`de`, `ovpn0` и т.д.), **не Ethernet**.
3. Выберите файл, нажмите **Загрузить**.
4. Повторите для каждого `.bat` файла.

### 7.2 OpenWrt: pbr + домены

#### Установка pbr

```bash
opkg update
opkg install pbr luci-app-pbr
```

#### Настройка в LuCI

1. **Services - Policy Routing** - включите сервис pbr.
2. В **Interface Gateways** убедитесь, что ваш VPN-интерфейс (`awg_de` или `ovpn0`) есть в списке.
3. Включите опцию **Use resolver set support** (маршрутизация по доменам через dnsmasq).
4. В разделе **Policies** добавьте правила:

| Имя | Destination address | Interface |
|---|---|---|
| youtube | `youtube.com googlevideo.com ytimg.com` | awg_de |
| telegram | `telegram.org t.me` | awg_de |
| openai | `openai.com chatgpt.com` | awg_de |

5. **Save & Apply - Restart**.

## Шаг 8. Настройка DNS

DNS провайдера может подменять адреса нужных сервисов до того, как трафик дойдёт до туннеля. Чтобы этого избежать, нужно привязать DNS нужных доменов к VPN-интерфейсу.

### Keenetic

1. **Интернет-фильтры - Настройка DNS**.
2. В блоке **Серверы DNS** добавьте сервер:
   - Адрес: `1.1.1.1` (или `8.8.8.8`)
   - Подключение: ваш VPN-интерфейс (например, `de`).
3. Если в вашей версии KeeneticOS есть блок «DNS-маршруты», добавьте правила для доменов:
   - `*.youtube.com`, `*.googlevideo.com`
   - `*.telegram.org`
   - `*.openai.com`
   - DNS для каждого - через VPN-интерфейс.

### OpenWrt

1. **Network - DHCP and DNS**.
2. В поле **DNS forwardings** пропишите `1.1.1.1` и `8.8.8.8`.
3. pbr с включённым **resolver set support** сам привяжет DNS-запросы нужных доменов к VPN-интерфейсу.

| | Свой VPS | Amnezia Premium |
|---|---|---|
| Цена | от $2.10/мес | $4-4.67/мес |
| Настройка | 10 минут один раз | сразу работает |
| Свой IP | да | нет (общая инфраструктура) |
| Стран | 1 (где арендовали) | 20+ |
| Устройств | без ограничений | до 7 |

**Когда выгоднее свой VPS:**
- нужен один чистый IP на семью

**Когда выгоднее Premium:**
- не хотите думать про VPS вообще
- нужен формат «заплатил и сразу работает»
- важна поддержка большого числа стран.
